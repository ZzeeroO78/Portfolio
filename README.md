## 📊 02-data-analysis-app

### 📖 Opis
Profesionalna full-stack aplikacija za analizu podataka sa sistemom autentifikacije, upravljanjem korisnicima, role-based pristupom i naprednim vizualizacijama.

### ⭐ Glavne Funkcije
- JWT autentifikacija sa role sistemom (radnik, menadjer, vlasnik, admin)
- Real-time dashboard sa automatskim kalkulacijama
- Interaktivni grafikoni (Chart.js)
- Dark/Light mode
- Paginacija i sortiranje
- CSV Export
- Activity logging

### 🛠️ Tehnologije
- **Frontend**: React 18, Vite 5, Tailwind CSS 3, Chart.js
- **Backend**: Node.js, Express.js, SQLite
- **Auth**: JWT, bcryptjs

### 🚀 Pokretanje
```bash
cd data-analysis-app

# Backend
cd server && npm install && npm run dev

# Frontend (novi terminal)
cd client && npm install && npm run dev

# Otvori http://localhost:3000
```
